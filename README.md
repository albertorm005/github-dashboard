# 🚀 GitHub Repository Analytics Dashboard

Aplicación web desarrollada con **Java + Spring Boot** que permite analizar repositorios y usuarios de GitHub en tiempo real.

---

## ✨ Features

* 🔍 Repository analysis (stars, forks, watchers)
* 👤 User analysis (list of repositories)
* 📊 Interactive charts (Chart.js)
* 👥 Top contributors
* 📈 Commit activity (last year)
* 🧠 Language distribution
* 📥 Export data to JSON
* 📄 Export report to PDF
* 🕘 Search history
* 🌙 Dark mode UI

---

## 🛠️ Tech Stack

**Backend**

* Java 17
* Spring Boot
* REST API

**Frontend**

* Thymeleaf
* HTML / CSS / JavaScript
* Chart.js

**Other**

* GitHub REST API
* iText PDF

---

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/albertorm005/github-dashboard.git
cd github-dashboard
```

Run the application:

```bash
./mvnw spring-boot:run
```

Open in browser:

```
http://localhost:8081
```
